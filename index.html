# HTML Template for Interactive Web-Based Lesson with H5P Activities Spot the Fallacy: Unmasking Bad Faith Arguments

Welcome to Spot the Fallacy!

This interactive lesson will teach you how to identify common logical fallacies used in bad faith arguments.

What is a Logical Fallacy?

A logical fallacy is an error in reasoning that renders an argument invalid.

Common Logical Fallacies

Ad Hominem

Attacking the person instead of the argument.

Strawman

Misrepresenting someone's argument to make it easier to attack.
